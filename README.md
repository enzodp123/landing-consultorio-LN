# Landing Page - Consultorio de Kinesiología LN

Bienvenido al repositorio de la landing page para el Consultorio de Kinesiología. Este proyecto está desarrollado con **Astro** para garantizar un sitio web rápido, eficiente y optimizado para SEO.

## 🚀 Descripción

Este sitio web sirve como punto de contacto digital para el consultorio, permitiendo a los pacientes conocer los servicios ofrecidos, la ubicación y facilitar el contacto o reserva de turnos.

## 🛠️ Tecnologías

- **[Astro](https://astro.build):** Framework principal para la generación del sitio estático.
- **[Tailwind CSS](https://tailwindcss.com):** Framework CSS para estilos rápidos y personalizados.
- **HTML/CSS/JS:** Estándares web modernos.
- **PNPM:** Gestor de paquetes utilizado.

## 🧞 Comandos

Todos los comandos se ejecutan desde la raíz del proyecto en la terminal:

| Comando | Acción |
| :--- | :--- |
| `pnpm install` | Instala las dependencias del proyecto |
| `pnpm dev` | Inicia el servidor de desarrollo en `localhost:4321` |
| `pnpm build` | Compila el sitio para producción en la carpeta `./dist/` |
| `pnpm preview` | Previsualiza la compilación localmente antes de subirla |

## 📂 Estructura del Proyecto

```text
/
├── public/      # Archivos estáticos (imágenes, favicon)
├── src/
│   ├── components/  # Componentes reutilizables (Header, Footer, Cards)
│   ├── layouts/     # Plantillas de diseño base
│   └── pages/       # Rutas y páginas del sitio
└── package.json
```
